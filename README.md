# SVG Drawing Tool

**Course Outcome (CO1):** Explain the fundamental concepts of front-end and back-end development, including HTML, CSS, JavaScript, React.js, Node.js, and databases.

## Aim
To build an interactive SVG drawing tool with mouse event handlers.

## Objectives
- Create an SVG canvas component
- Implement mouse event tracking for drawing
- Add shape drawing functionality
- Include color selection and undo feature
- Make responsive with viewport units

## Hardware/Software Requirements
- **Hardware:** PC/Laptop, 4GB+ RAM, Mouse/Touchpad
- **Software:** 
  - Web Browser (Chrome, Firefox, Edge)
  - Code Editor (VS Code, Sublime Text, etc.)
  - Node.js (optional, if running a server)
  - Git (for version control)

## Folder Structure
svg-drawing-tool/
│
├─ index.html ← Main HTML file containing SVG canvas and toolbar
├─ style.css ← CSS styling for layout and responsiveness
├─ script.js ← JavaScript functionality (drawing, color selection, undo)
└─ README.md ← Project documentation (this file)
